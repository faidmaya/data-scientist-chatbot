# Data Scientist Productivity Assistant

Chatbot berbasis AI menggunakan Gemini API untuk membantu data scientists dengan tips ML, Python, dan rekomendasi dataset.

## Use Case
- Asistensi produktivitas: Jawab pertanyaan tentang data science, generate ide kode, dan rekomendasikan dataset.

## Parameter Kreatif
- Gaya bahasa: Santai dan ramah (misalnya, "bro", emoji).
- Domain: Data science (ML, Python, statistik).
- Fitur: Memory (ingat konteks), rekomendasi dataset via integrasi sederhana.

## Cara Run
1. Install dependencies: `python3 -m pip install -r requirements.txt`
2. Setup `.env` dengan GEMINI_API_KEY.
3. Jalankan: `python3 -m streamlit run app.py`
